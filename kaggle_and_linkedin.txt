## About This Dataset

This dataset contains **43,872 rows** of hourly weather observations for **Lahore, Pakistan** spanning **5 years (2020–2025)**, collected via the Open-Meteo Archive API.

Lahore is one of Pakistan's most populated cities and experiences extreme seasonal variation — scorching summers above 40°C, cold winters, and heavy monsoon rains. This dataset captures all of it in hourly detail.

---

## What's Inside

- **Temperature** — actual and feels-like temperature (°C)
- **Humidity** — relative humidity (%)
- **Precipitation** — hourly rainfall (mm)
- **Wind Speed** — wind speed at 10m height (km/h)
- **Weather Conditions** — WMO weather codes with human readable descriptions
- **Time Features** — year, month, day, hour, season extracted for easy analysis

---

## Potential Use Cases

- Time series forecasting (predicting temperature/rainfall)
- Climate trend analysis for Lahore
- Seasonal pattern analysis
- Weather classification models
- EDA practice for beginners

---

## Data Source

- **API:** Open-Meteo Archive API (open-meteo.com)
- **Location:** Lahore, Pakistan — 31.5497°N, 74.3436°E
- **Elevation:** 222m
- **Timezone:** Asia/Karachi (GMT+5)
- **Frequency:** Hourly
- **License:** CC0 Public Domain

---

## Methodology

1. Fetched raw JSON data from Open-Meteo Archive API
2. Extracted hourly data into a pandas DataFrame
3. Checked and handled null values and duplicates
4. Converted time column to datetime format
5. Extracted time features (year, month, day, hour)
6. Added season and weather description columns
7. Exported final clean CSV
